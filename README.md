# HASHTTORY
Hashttory is a digital historiography prototype that explores how hashtags function as temporal inscriptions in cyberspace. It aims to map, analyse and interpret this  inscriptions to broaden historical narratives in the digital age.
Objective
To identify, document and contextualise historically relevant hashtags that serve as markers of time, memory and struggle.
Theorical Framework
Based on concepts by Fernand Braudel, Reinhart Koselleck, François Hartog, and Pierre Lévy.
Tchnologies Used
HTML, CSS, JavaScript
Git & GitHub for version control

Structure
'/images': illistrations and logos
'/css':style sheets
'/index.html':homepage with hashtag entries

Author
